# ConnectSphere

**ConnectSphere** is a faith-inspired professional networking and growth platform designed to help young people connect, collaborate, learn, and discover opportunities that align with their skills, interests, career goals, and personal development journey.

# 📖 Overview

Many young people struggle to find mentors, opportunities, project collaborators, and supportive communities that align with their goals.

ConnectSphere addresses this challenge by providing a centralized platform where users can:

* Build professional profiles
* Discover communities
* Collaborate on projects
* Access opportunities
* Connect with mentors
* Receive AI-powered growth recommendations
* Track personal development progress

The platform combines networking, mentorship, collaboration, and career growth into a single ecosystem.

---

# 🎯 Problem Statement

Young people often face:

* Limited access to mentors
* Difficulty finding relevant opportunities
* Fragmented learning communities
* Lack of professional networking platforms tailored to growth and purpose
* Limited guidance on career development

As a result, many talented individuals struggle to reach their full potential.

# 👨‍💻 Installation

Clone the repository:

```bash
git clone https://github.com/neduojile/Connectsphere.git
```

Navigate into the project:

```bash
cd Connectsphere
```

Install dependencies:

```bash
npm install
```

Configure environment variables:

```env
DATABASE_URL=
OPENAI_API_KEY=
NEXTAUTH_SECRET=
```

Run development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
